# Getting setup 

- Create a directory on your computer called streamerbot-leaderboards
- Create a textfile called raids.json inside of the streamerbot-leaderboards directory
- Edit the raids.json file and put just ``{}`` in the file save and close the file


# Create the Streamerbot Action to catch the Raid events

- Go to Actions
- Right-click the action list
- Select Add
- Name The Action ``Raid Leaderboard - Record Raid``
- Open the New Action 
- Add a Sub-Action ``Core -> C# -> Execute C# Code
