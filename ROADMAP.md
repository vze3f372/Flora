# Flora Roadmap

## Current status

Flora is now a local-first OBS/Streamer.bot panel engine with:

- table leaderboards
- goal panels
- recent activity panels
- default and named rotations
- local Admin UI
- Streamer.bot Fetch URL integration
- Twitch avatar cache
- server launcher scripts
- runtime reset and restore
- config/goals backup management
- admin setup presets
- panel theme presets
- panel layout presets
- OBS source URL quick-copy tools
